# Avance-del-proyecto-backend-songs
# Backend Songs – Proyecto Final (MVP)

Backend desarrollado con Node.js y Express para gestionar canciones usando operaciones CRUD.
Los datos se manejan en memoria mediante arreglos.

## Cómo ejecutar el proyecto
1. npm install
2. node index.js

## Endpoints
GET /songs  
POST /songs  
PUT /songs/:id  
DELETE /songs/:id
